# Language Service and VS Code

The AXSG language service understands the compiler semantics behind:

- compiled bindings
- shorthand C# expressions
- inline C# blocks and CDATA
- selector/property/theme navigation
- XAML-aware rename and cross-language references

The VS Code extension hosts that language server and augments it with projected C# interop where it adds real value.

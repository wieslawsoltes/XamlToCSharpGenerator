# XamlToCSharpGenerator

XamlToCSharpGenerator compiles Avalonia XAML into generated C# and layers project tooling around that compiler:

- compiled bindings and expression bindings
- inline C# in valid XAML
- hot reload and hot design runtime integration
- VS Code language server and extension support
- runtime fallback services and editor integrations

Use the docs sections for setup, language features, architecture, and release workflows.
